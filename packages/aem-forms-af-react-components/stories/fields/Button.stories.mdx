# Button

The button component can be created using the `button` fieldType

```json
{
  "fieldType": "button",
  "label" : {
    "value": "..."
  }
}
```

## Click Event

Buttons can specify an
[event handler](https://git.corp.adobe.com/pages/livecycle/af2-docs/spec/latest/index.html#_event_handler) on
[click](https://git.corp.adobe.com/pages/livecycle/af2-docs/spec/latest/index.html#event-click) event using
[JSON Formula](https://git.corp.adobe.com/pages/livecycle/af2-docs/spec/latest/index.html#_expression_grammar)
Expressions

The example below demonstrates changing the label of the button upon click
```
{
  "fieldType": "button",
  ...
  "events": {
    "click": "{label: {value : 'some value' }}"
  }
}
```


<Story
    name="Click"
args={{
    formJson: form({
        fieldType: 'button',
        label : {
            value : "Click Me"
        },
        events : {
            "click": "{label:{value : if(startsWith($label.value, 'Click'), 'You Clicked me', 'Click Me Again')}}"
        }
    })
}} >{Template.bind({})}</Story>

## Submit

The common functionality of a button is to submit the form. This can be achieved by using the
[submitForm](https://git.corp.adobe.com/pages/livecycle/af2-docs/spec/latest/index.html#submit_form_function) function
on click and specifying the `action` property on the form

```
{
  "action": "https://customdomain.com",
  "items" : [
    ...,
    {
      "fieldType": "button",
      ....
      "events" : {
        "click" : "submitForm()"
      }
    }
  ]
}
```

<Story
    name="Submit"
    args={{
        formJson: {...form({
            fieldType: 'button',
            label : {
                value : "I submit the form to https://www.google.com/"
            },
            events : {
                "click": "submitForm()"
            }
        }),
        action: "https://www.google.com/"
        }
    }} >{Template.bind({})}</Story>
