# Form Components

The Form specification provides different field types which are represented using the
[viewType](https://git.corp.adobe.com/pages/livecycle/af2-docs/spec/latest/#field-properties-viewType) property in
the JSON

## Checkbox

Checkbox component can have an on or an off value. When selected the on value is submitted while upon deselection
off value is submitted.
The values are provided via the
[enum](https://git.corp.adobe.com/pages/livecycle/af2-docs/spec/latest/#field-properties-enum) property. The first
value in the array is an on value, while the second is off value. Any other value will be discarded

<Story id="crispr-form-components-checkbox--def" />

The off value is optional, if not defined then no value will be set

<Story id="crispr-form-components-checkbox--no-off-value" />

A checkbox can specify enum implicitly by setting
[type](https://git.corp.adobe.com/pages/livecycle/af2-docs/spec/latest/#field-properties-type) property to boolean

<Story id="crispr-form-components-checkbox--boolean" />

## Checkbox Group

Checkbox Group behaves similar to a multi selection list

<Story id="crispr-form-components-checkbox-group--def" />

It also supports single selection, by setting the type property to not be an array type (i.e. string, boolean or number)
depending upon the type of values in enum

<Story id="crispr-form-components-checkbox-group--single-selection" />

## Date Field

A Date Field is a component to capture a date that user can enter using keyboard or
use the widget to select a number. The date is serialized using ISO 8601 format (YYYY-MM-DD)

<Story id="crispr-form-components--date-input" />

## Drop Down

Drop Down allows selecting a value from a set of options. The option are provided via the
[enum](https://git.corp.adobe.com/pages/livecycle/af2-docs/spec/latest/#field-properties-enum) property.

Note: The Drop-Down component currently supports only single selection.

<Story id="crispr-form-components-dropdown--def"/>

The label of the options can be configured by providing
the [enumNames](https://git.corp.adobe.com/pages/livecycle/af2-docs/spec/latest/#field-properties-enumNames) property.

<Story id="crispr-form-components-dropdown--enum-names"/>
