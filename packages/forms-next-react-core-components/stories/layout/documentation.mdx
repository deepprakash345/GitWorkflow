# Form Layouts

Crispr Out of the Box does not support any layouts. But when the form is rendered on the screen a layout is required.
Adaptive Form Super Component provides the layout functionality via custom View Types and custom Property that
Specification Supports.

The React based AF Super Component supports the following layouts
